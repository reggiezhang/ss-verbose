## ss-verbose
Based on [shadowsocks-libev-3.1.3.tar.gz](https://github.com/shadowsocks/shadowsocks-libev/releases/download/v3.1.3/shadowsocks-libev-3.1.3.tar.gz)
## Features
- ss-local
    + local dns cache
    + outbound block list
    + enable syslog by -w
- ss-server
    + log client ip in verbose mode
    + remove ipv6 dns lookup
