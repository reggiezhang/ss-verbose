# ss-verbose

## Intro

Based on [shadowsocks-libev](https://github.com/shadowsocks/shadowsocks-libev)


Current version: 3.1.0 | [Changelog](debian/changelog)

## Features
- ss-local
    + local dns cache
    + outbound block list
    + enable syslog by -w
- ss-server
    + log client ip in verbose mode
    + remove ipv6 dns lookup
